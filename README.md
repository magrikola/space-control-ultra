# Space Control Ultra

This is a turn-based strategy game developed using Python and Pygame.

The game is played on a 4x4 grid where a human player competes against an AI agent.

## AI Algorithm
The AI uses the Minimax algorithm to evaluate possible moves and choose the best one.

## Features
- Player vs AI gameplay
- Minimax decision-making
- Grid-based UI
- Restart system

python main.py
